


# 📝 To-Do List Web App

This is a simple and clean To-Do List web app built using HTML, CSS, and JavaScript.

## ✅ Features
- Add task 🟢  
- Mark task as completed ✅  
- Delete task ❌  
- Simple, clean layout  
- Responsive design

## 🧠 Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla)
